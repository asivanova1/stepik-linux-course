## Вопрос 19

**Формулировка вопроса:** Напишите скрипт на bash, который будет искать наибольший общий делитель (НОД) двух чисел.

**Ответ:** 
```bash
#!/bin/bash
gcd() {
    if [[ $1 -eq $2 ]]; then echo "GCD is $1"
    elif [[ $1 -gt $2 ]]; then gcd $(($1 - $2)) $2
    else gcd $1 $(($2 - $1))
    fi
}
while read m n; do
    if [[ -z "$m" ]]; then echo "bye"; break; fi
    gcd $m $n
done
```

**Пояснение:** Алгоритм Евклида через вычитание.

## Вопрос 20

**Формулировка вопроса:** Напишите калькулятор на bash.

**Ответ:** 
```bash
#!/bin/bash
while read a op b; do
    if [[ "$a" == "exit" ]]; then echo "bye"; break
    elif [[ -z "$a" || -z "$op" || -z "$b" ]]; then echo "error"; break
    else
        case $op in
            "+") echo $(($a + $b));;
            "-") echo $(($a - $b));;
            "*") echo $(($a * $b));;
            "/") echo $(($a / $b));;
            "%") echo $(($a % $b));;
            "**") echo $(($a ** $b));;
            *) echo "error"; break;;
        esac
    fi
done
```

**Пояснение:** Проверка на три аргумента и неизвестную операцию.
