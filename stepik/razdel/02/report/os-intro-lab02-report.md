# Цель раздела

Изучить основы работы с удалёнными серверами: подключение по SSH, передачу файлов, управление процессами, использование tmux и работу с биоинформатическими программами (FastQC, Clustal, bowtie2).

## Вопрос 1

**Формулировка вопроса:** Для каких задач можно использовать удаленный сервер?

**Ответ:** 
- Хранение конфиденциальных данных
- Хранение общедоступных данных
- Хранение больших объемов данных
- Выполнение сложных (затратных по памяти и времени) вычислений

**Пояснение:** Удалённый сервер может использоваться для любых задач, связанных с хранением данных (как конфиденциальных, так и общедоступных) и выполнением ресурсоёмких вычислений.

## Вопрос 2

**Формулировка вопроса:** Предположим программа ssh-keygen создала вам два ключа: id_rsa и id_rsa.pub. Какой из этих ключей можно без опаски пересылать по интернету?

**Ответ:** id_rsa.pub

**Пояснение:** Публичный ключ (id_rsa.pub) можно свободно распространять. Приватный ключ (id_rsa) должен храниться в секрете и никогда не передаваться.

## Вопрос 3

**Формулировка вопроса:** Какая команда скопирует на сервер (в домашнюю директорию) папку stepic вместе с содержимым ее самой и всех ее подпапок?

**Ответ:** scp -r stepic username@server:~/

**Пояснение:** Опция `-r` (recursive) позволяет копировать директории рекурсивно со всем содержимым.

## Вопрос 4

**Формулировка вопроса:** Предположим, что вы устанавливаете программу program на свой компьютер при помощи команды sudo apt-get install program. Терминал сообщает вам, что он не может найти и скачать установочный пакет. Какие действия могут устранить проблему?

**Ответ:** 
- Проверка места на диске и его очистка, если диск переполнен.
- Проверка интернет соединения и его установка, если соединения нет.

**Пояснение:** Основные причины — отсутствие интернета или недостаточно места на диске. Команды upgrade и --only-upgrade не помогут, если пакет не найден в репозитории.

## Вопрос 5

**Формулировка вопроса:** Для чего можно использовать программу Filezilla?

**Ответ:** 
- Для копирования файлов со своего компьютера на сервер
- Для копирования файлов с сервера на свой компьютер
- Для просмотра содержимого директорий на сервере

**Пояснение:** Filezilla — FTP-клиент для передачи файлов и просмотра содержимого удалённых директорий, но не для запуска программ или просмотра локальных директорий (это делает файловый менеджер ОС).

## Вопрос 9

**Формулировка вопроса:** Посмотрите справку по программе Clustal и впишите в поле ниже команду, которая запускает в терминале Clustal на файле test.fasta и выполняет множественное выравнивание (multiple alignment).

**Ответ:** `clustalw -ALIGN -INFILE=test.fasta` (или `clustalw2 -ALIGN -INFILE=test.fasta`)

**Пояснение:** Опция `-ALIGN` явно указывает на выполнение множественного выравнивания. Файл с последовательностями передаётся через `-INFILE`.

**Формулировка вопроса:** jobs, top и ps позволяют отслеживать работу запущенных в терминале программ. В каждой из этих трех утилит для каждой запущенной программы указывается число-идентификатор. Одинаковые ли эти идентификаторы в jobs, top и ps?

## Вопрос 12

**Формулировка вопроса:** С помощью какой команды можно мгновенно завершить остановленный процесс?

**Ответ:** kill -9

**Пояснение:** Сигнал SIGKILL (-9) безусловно завершает процесс без возможности его перехвата или игнорирования.

## Вопрос 13

**Формулировка вопроса:** Что произойдет, если использовать kill (без опций) по отношению к процессу, который был приостановлен при помощи Ctrl+Z?

**Ответ:** Процесс приступит к завершению, как только будет продолжен

**Пояснение:** SIGTERM (сигнал по умолчанию) не действует на остановленный процесс, пока он не будет продолжен командой `fg` или `bg`.

## Вопрос 14

**Ответ:** 0% CPU

**Пояснение:** Остановленный процесс не использует процессорное время, он просто находится в памяти в ожидании продолжения.

## Вопрос 24

**Формулировка вопроса:** Изучите работу с "вкладками внутри вкладок" и отметите верные утверждения из списка ниже.

**Ответ:** 
- Если разделенную горизонтально вкладку разделить еще и вертикально (т.е. нажать один раз Ctrl+B и %), то получится 4 одинаковые "части"
- Команды-"разделения" действуют только в текущей вкладке tmux, а не во всех вкладках одновременно
- Если набрать в одной из "частей" вкладки команду exit, то вся вкладка закроется

**Пояснение:** Разделение работает рекурсивно (2×2=4 части), действует только на текущую вкладку, а `exit` закрывает всю вкладку целиком.
